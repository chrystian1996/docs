======================
**Periodo Financiero**
======================

Solop cuenta con el mantenimiento de Calendario, Año y Período para
poder controlar su contabilidad.

En cada Calendario se podría definir los Años junto con sus períodos
correspondientes. De esta manera podrá controlar el Estado de un
periodo, pudiendo el mismo estar "Abierto", "Cerrado" o "Cerrado
Permanentemente" (opción poco recomendada dado a que no se podrá
revertir en caso de error). A su vez, dentro de cada Período, se contará
con un control por "Tipo de Documento Base", permitiendo en caso de ser
necesario, poder cerrar períodos de manera parcial, dejando ciertos
Documentos Abiertos para que si puedan seguir siendo generados.
