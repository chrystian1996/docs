===============================
**Nivel de Línea del Proyecto**
===============================

**Tamaño del Proyecto (Nivel de Línea)**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Los diferentes niveles posibles son: **Proyecto, Fase, Tarea.**

Un Proyecto podrá tener diferentes niveles según su complejidad o
tamaño. Dentro de cada Nivel seleccionado, se contará con una pestaña de
Solicitudes y una pestaña de Líneas. Dentro de las líneas se podrá
definir cada uno de los Productos que se estará manejando, ya sea para
que luego se facture un servicio brindado por la Agencia y que deba ser
Facturado al Cliente, como también un Producto que deba cotizarse a
proveedores dentro de las Tareas (Cotizaciones a realizar).
